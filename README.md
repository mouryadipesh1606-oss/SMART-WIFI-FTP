# 🌐 AirBridgeX | WiFi FTP System

A Python-based FTP system that allows file sharing between devices connected to the same WiFi network. Includes both CLI and GUI versions for better usability.

## 🚀 Features

- Share files over local WiFi network  
- GUI-based FTP manager  
- CLI support with arguments  
- Custom port and directory selection  
- User authentication support  

---

## 🛠️ Tech Stack

- Python  
- pyftpdlib  
- Tkinter (GUI)  

---

## 📁 Project Structure
wifi_ftp_gui.py → GUI-based FTP manager
wififtp.py → CLI FTP system
pyproject.toml → project configuration

---

## ▶️ How to Run

### GUI Version
```bash
python wifi_ftp_gui.py

---

## CLI Version
python wififtp.py

---

##🌐 Example FTP URL
ftp://192.168.x.x:2121

##📌 Use Case
Useful for quick file sharing between devices connected to the same network without internet.
